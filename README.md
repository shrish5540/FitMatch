# 🔥 FitMatch Backend

Backend API for FitMatch — an AI-powered gym partner finder with swipe matching, real-time chat, AI fitness coaching, workout generation, diet recommendations, and progress tracking.

## 🚀 Features

- JWT Authentication
- Swipe-Based Matching
- Real-Time Chat with Socket.IO
- AI Fitness Coach
- AI Workout Generator
- AI Diet Recommendations
- Progress Tracking
- Leaderboard System
- Cloudinary Image Uploads

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Socket.IO
- Cloudinary
- Groq AI

## ⚙️ Environment Variables

Create a `.env` file:

```env
PORT=
MONGO_URI=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
GROQ_API_KEY=
```

## ▶️ Run Locally

```bash
npm install
npm run dev
```

## 🌐 Deployment

- Backend: Render
- Database: MongoDB Atlas
